<objective>
Add the opt-in, OFF-by-default saved-collection sync (CAP-05): when explicitly enabled, enumerate a user's Instagram saved collection into reel URLs via gallery-dl, then feed each URL through the SAME deduped submit -> queue path the worker already drains (D-08). Purpose: Let the user opt in to bulk-capturing a saved collection without building a parallel pipeline and without tripping Instagram's rate/pattern detection. Honors D-08 (opt-in, cookie-based, small batches with jitter) and the OPS ban-risk caution.
Output: A gallery-dl enumeration + jitter/batch sync module, a runnable `sync` command, and tests for the off-by-default guard, dedup, and graceful failure.
</objective>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user-configured collection URL + browser -> gallery-dl subprocess | config-supplied URL/browser cross into a subprocess invocation |
| Instagram (remote) -> enumerated URL list | untrusted remote output (reel URLs) crosses back into the submit path |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Tampering | gallery-dl invocation (saved-sync.ts) | mitigate | `Bun.spawn([...])` arg array, never a shell string (Task 1 grep gate) |
| T-03-02 | Tampering | enumerated reel URLs -> submit | mitigate | Each URL re-validated via parseReelShortcode before insert/enqueue (same guard as URL intake) |
| T-03-03 | Denial of Service | worker stability on gallery-dl failure | mitigate | try/catch around the run; 572/cookie-lock logged + run halts gracefully, never rethrown into the worker loop (Pitfall 5) |
| T-03-04 | Denial of Service | Instagram account ban from request velocity | mitigate | Off by default; capped batch (SYNC_BATCH_SIZE) + randomized jitter delays; no fixed interval (RESEARCH §2) |
| T-03-05 | Information Disclosure | cookie handling / logs | mitigate | Cookies read from the browser at call time by gallery-dl (not stored/logged); never log the cookie browser/env wholesale |
| T-03-SC | Tampering | npm/pip/cargo installs | accept | No new npm packages; gallery-dl is a user-installed PyPI tool [VERIFIED: PyPI] in RESEARCH Package Legitimacy Audit |
